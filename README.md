# songKakasi

***

Develop with MacOS  
Base on Python3  

## 依赖包

- requests
- tkinter
- json
- sys
- os
- io
- gTTS
- pykakasi

> gTTS lib需科学上网

MacOS系统下需安装SoX(目前只支持MacOS，之后适配其他系统):  

```bash
$ brew install sox
```

## 使用方法

```bash
$ python3 kakasi.py       # 调出GUI界面
$ python3 kakasi.py カカシ # 直接转换并读出 
```

其中GUI界面中songID指的是网易云歌曲ID
